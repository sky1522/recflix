# RecFlix — 헤더 기분 드롭다운 너비 확장

## 문제
헤더의 기분 선택 드롭다운에서 "몽글몽글한", "상상에 빠진" 라벨이 줄바꿈되어 2줄로 표시됨. 나머지 6개(평온한, 긴장된, 활기찬, 유쾌한, 울적한, 답답한)는 3글자라 정상.

## 수정
기분 드롭다운의 너비를 확장하여 5글자 라벨도 한 줄에 표시되도록 한다.

## 수정 파일
헤더에서 기분 드롭다운을 렌더링하는 컴포넌트를 찾아서 드롭다운 컨테이너의 너비 또는 그리드 컬럼 너비를 확장한다. 2×4 그리드 구조는 유지하되 각 셀이 "몽글몽글한"(5글자 + 이모지)을 한 줄에 담을 수 있는 너비로 조정.

현재 드롭다운 너비가 약 200px 수준으로 보이는데, 260~280px 정도로 확장하면 충분할 것. `min-w` 또는 `w-` 클래스를 조정.

모바일 드로어(HeaderMobileDrawer.tsx)에도 동일한 기분 그리드가 있다면 함께 확인하되, 모바일은 화면이 넓어서 문제 없을 수 있음.

## 검증
1. 데스크톱: 헤더 기분 드롭다운에서 8개 라벨 모두 한 줄 표시 확인
2. 모바일(360px): 드로어에서 기분 그리드 깨지지 않는지 확인
3. npx next build 성공 확인

## 완료 후
1. git add -A && git commit -m "fix: 기분 드롭다운 너비 확장 (5글자 라벨 줄바꿈 해소)"
2. git push origin main
3. 결과를 claude_results.md에 저장
