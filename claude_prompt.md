# 수정 2건: 한국 인기 영화 섹션 일관성 + 히어로 유도 섹션 UI 개선

## 수정 1: 한국 인기 영화 섹션 — 다른 섹션과 일관성 맞추기

### 문제
한국 인기 영화 섹션이 다른 추천 섹션(인기, 높은평점, 날씨, 기분, MBTI)과 UI가 다름:
- ❌ 섹션 타이틀 앞에 이모지 없음 (다른 섹션은 🔥, ⭐ 등 이모지가 있음)
- ❌ 섹션 우측에 새로고침(🔄) 버튼 없음

### 작업
1. **기존 섹션 패턴 분석 먼저** — 다른 MovieRow 섹션(인기, 높은평점, 날씨추천 등)의 title, subtitle, 이모지, 새로고침 버튼이 어떻게 구현되어 있는지 확인
2. **동일 패턴 적용**:
   - 타이틀: `"🇰🇷 한국 인기 영화"` (이모지 포함)
   - 서브타이틀: 기존 큐레이션 패턴과 동일하게
   - 새로고침 버튼: 기존 섹션의 🔄 버튼과 동일한 컴포넌트/핸들러 사용
   - 새로고침 시 `getMovies({ country: "대한민국", sort_by: "popularity" })` 재호출 + 랜덤 셔플 또는 페이지 변경
3. **다른 섹션과 100% 동일한 레이아웃/스타일**이어야 함. 차이가 있다면 맞춰서 수정

## 수정 2: 히어로 섹션 우측 상단 유도 섹션(MBTI/날씨/기분) UI 개선

### 현재 상태 확인 (먼저)
- `frontend/app/page.tsx` 또는 히어로 관련 컴포넌트에서 MBTI, 날씨, 기분 유도 UI가 어떻게 렌더링되는지 확인
- 현재 박스 크기, 글씨 크기, 패딩, 간격 확인
- 모바일에서의 렌더링 상태 확인 (반응형 클래스 유무)

### 개선 방향

**데스크탑:**
- 유도 박스(MBTI, 날씨, 기분 등) 크기와 패딩을 충분히 확보하여 클릭하기 편하게
- 글씨 크기가 너무 작다면 적절히 키우기 (가독성 확보)
- 박스 간 간격(gap) 일정하게
- hover 효과 개선 (배경색 변화, 살짝 스케일 등으로 클릭 가능함을 표현)
- 선택된 상태와 미선택 상태의 시각적 구분 명확하게

**모바일:**
- 터치 타겟 최소 44px 확보 (Apple HIG 기준)
- 박스가 너무 작아서 손가락으로 선택하기 어려우면 안 됨
- 텍스트 잘림 없이 완전히 표시
- 가로 스크롤 또는 줄바꿈으로 모든 옵션이 접근 가능해야 함
- 모바일에서 유도 섹션이 히어로 이미지를 가리지 않도록 레이아웃 조정

**공통:**
- 현재 박스/글씨가 너무 작거나 밀집되어 있다면 → 여유 있는 크기로 조정
- 각 유도 항목(MBTI, 날씨, 기분)의 아이콘/이모지가 있다면 크기도 함께 조정
- 선택 시 피드백 (배경색 변경, 체크마크 등)

### 구체적 스타일 가이드 (참고)
```
// 데스크탑 유도 박스
- 최소 높이: 40px
- 패딩: px-4 py-2 이상
- 글씨: text-sm 이상 (12px 미만 금지)
- 간격: gap-2 또는 gap-3
- border-radius: rounded-lg
- hover: bg-white/20 또는 scale-105 transition

// 모바일 유도 박스
- 최소 높이: 44px (터치 타겟)
- 패딩: px-3 py-2.5 이상
- 글씨: text-sm 이상
- 가로 overflow-x-auto 스크롤 허용
```
